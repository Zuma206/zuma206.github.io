gVX

Press the call button and enter a filename to call. This will enable the called file buttons. The icon button askes for the name of an icon, and will set that as the called exe's icon. Bind will ask for a filename to bind to the called exe, and the name of the resultant exe. Manifest will edit the manifest of the called file to make it require admin rights in order to run
